Wine 在 Ubuntu 下的优异表现

Jan 26th, 2008 12:00 am | Comments

Wine 这个软件功能越来越强大了，非常好用～～～

源里面的 Wine 版本太旧，正好在昨天，Wine 发布了 0.9.54 ，可以直接去官方网站http://www.winehq.org/site/download下载：

首先要获取密钥：

 wget -q http://wine.budgetdedicated.com/apt/387EE263.gpg -O- | sudo apt-key add -

然后下载源列表：

For Ubuntu Gutsy (7.10):

sudo wget http://wine.budgetdedicated.com/apt/sources.list.d/gutsy.list -O /etc/apt/sources.list.d/winehq.list

For Ubuntu Feisty (7.04):

sudo wget http://wine.budgetdedicated.com/apt/sources.list.d/feisty.list -O /etc/apt/sources.list.d/winehq.list

随后执行更新并下载：

sudo apt-get update
sudo apt-get install wine

下载完毕后处理中文问题：

从 Windows 下拷贝 SimSun.ttf 字体到~/.wine/drive_c/windows/fonts/下，将权限改为 755。

编辑下面一段文字：

REGEDIT4

[HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\FontSubstitutes]
"Arial"="simsun"
"Arial CE,238"="simsun"
"Arial CYR,204"="simsun"
"Arial Greek,161"="simsun"
"Arial TUR,162"="simsun"
"Courier New"="simsun"
"Courier New CE,238"="simsun"
"Courier New CYR,204"="simsun"
"Courier New Greek,161"="simsun"
"Courier New TUR,162"="simsun"
"FixedSys"="simsun"
"Helv"="simsun"
"Helvetica"="simsun"
"MS Sans Serif"="simsun"
"MS Shell Dlg"="simsun"
"MS Shell Dlg 2"="simsun"
"System"="simsun"
"Tahoma"="simsun"
"Times"="simsun"
"Times New Roman CE,238"="simsun"
"Times New Roman CYR,204"="simsun"
"Times New Roman Greek,161"="simsun"
"Times New Roman TUR,162"="simsun"
"Tms Rmn"="simsun"

保存为 zh.reg，在终端下通过 regedit 导入：

regedit zh.reg

然后就可以开始安装 Win 下的软件了：

范例一：安装迅雷国际版

在终端下定位到迅雷国际版的路径，执行

wine Gigaget.exe

像在 Windows 下安装一样，一路 Next 到底即可。

范例二：安装 CS

首先仍旧 wine CS.exe，一路 Next 到底，然后进入游戏界面后可能会出现中文不能显示的问题，此时请回到上文，仔细按照“处理中文”的步骤操作，注意 SimSun.ttf 的权限需要改成 755 或 777；开始游戏后会提示安装 Gecko，这个软件是 wine 下模拟 IE 的一种方式，此时可以联网下载，如果从官网下载速度慢也可以先退出游戏，上网搜 wine_Gecko-0.1.0 的安装文件，下载后将 wine_gecko-0.1.0.cab 复制到~/.wine/drive_c 下，在终端打开 regedit，定位到HKEY_CURRENT_USER/Software/Wine/MSHTML/，在右侧窗口修改GeckoUrl值为：C:\wine_gecko-0.1.0.cab，然后重新进入游戏，在提示此窗口时执行 Install 即可马上安装完毕。

范例三：安装 ies4linux

Gecko 是一个浏览器核心，在软件中需要显示 HTML 页面的时候经常使用，比如超星阅读器或者 CS，而要想更好地模拟 IE，就需要使用 ies4linux 这个软件，可以到http://www.tatanka.com.br/下载最新版的 ies4linux，下载后解压缩到某处，执行./ies4linux，按照提示一步步回答若干问题之后，就开始联网下载安装所需的文件，下载完毕后就可以在桌面上看到 ies4linux 的图标，打开后修改安全权限为低，发现能够登录支付宝了。

因为 ies4linux 的 wine 配置和系统中的 wine 配置是不同的，所以需要重新修改配置才能将 ies4linux 的中文显示问题完全解决：

gedit ~/.ies4linux/ie6/system.reg

修改这两行：

"MS Sans Serif"="sserife.fon"
"MS Serif"="serife.fon"

分别替换为： "MS Sans Serif"="SIMSUN.TTC"
"MS Serif"="SIMSUN.TTC"

另外修改如下代码：

[Software\Microsoft\Windows NT\CurrentVersion\FontSubstitutes]
1144897563

"MS Shell Dlg"="SimSun"
"MS Shell Dlg 2"="SimSun"

范例四：安装极品飞车 9

不知道是我的集成显卡太差还是没有真正安装成功的原因，安装好了之后本该有文字的地方都变成方块了，只好卸载掉……

安装方法：

最好是 DVD 英文版光盘，将光盘中所有内容复制到一个临时文件夹中，为避免安装过程中提示没有 DirectX9.0c，需要修改 AutoRun 的 Autorun.cfg 中其中两行的内容为：

DirectXVersion=0
StartMenuDXEULA=0

然后执行安装，安装结束后，从 Windows 分区的 System32 文件夹中复制或者到此地址http://www.dll-files.com/dllindex/dll-files.shtml?d3dx9_26下载d3dx9_26.dll，放到.wine/drive_c/windows/system32下。

上网下载极品飞车 9 的 1.3 版补丁和免 CD 补丁，覆盖即可。
