Linux学习笔记 (1)

Feb 3rd, 2008 12:00 am | Comments

不断更新中……

Q: 如何挂载 ISO 镜像到光驱（即虚拟光驱）
A: sudo mount -t iso9660 -o loop ./cdimg.iso /mnt/test
Q: 如何更改文件为可执行文件？何时需要更改？
A: 输入 sudo chmod a+x filename 即可将文件改为可执行文件。当用文本格式编写了脚本之后，可以执行这个操作执行脚本。
Q: 如何在终端下执行可执行文件？
A: ./filename 即可。
Q: 如何安装软件：
A: 软件的安装有两种，一种是二进制包安装，一种是源码安装。
二进制包可以直接用./filename 执行，按照提示逐步操作即可。
对于 Ubuntu Linux，可以直接执行 deb 包以安装，命令行下用 sudo dpkg -i XXX.deb 或者 sudo gdebi XXX.deb
对于 Redhat/Fedora、Suse 等操作系统，可以直接执行 rpm 包安装，命令行下用 rpm -i XXX.rpm
如 Adobe Flash Player 之类 tar.gz 的压缩包，应先解压，寻找类似于 install、installer 等文件，仍旧使用./filename 安装。（注意，虽然也是 tar.gz 压缩包，但是有别于源码包）
对于源码包，通常使用“安装三部曲”来进行：
配置安装环境，生成 Makefile：
./configure （可带参数--prefix，为定义安装路径，如：./configure –prefix /home/caspar/google-earth）
编译链接生成可执行文件、类库等目标文件：
make
将生成的文件复制到对应的路径以安装：
make install
另外，一般的软件包解压缩后都可以找到一个 README 文件，可以按照文件里的步骤安装。
