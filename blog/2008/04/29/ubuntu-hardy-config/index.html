Ubuntu 8.04 Config

Apr 29th, 2008 12:00 am | Comments

注：本文并非为纯新手准备，请在看本文前至少了解一些命令行操作并最好对Ubuntu以前版本或其他发行版的图形界面操作比较熟悉。如果你需要一些更初级的内容请访问下列页面：

Ubuntu 7.10 配置参考

Ubuntu 8.04 (Hardy Heron)果然没有让我失望，X2300的特效已经可以开启了，色偏，分辨率默认低的问题都完美解决了。

闲话少叙，直接开始配置吧:)

★更新

请根据您的网络环境和网速条件选择以下合适的一组源用以获取服务器软件包资源以及更新系统。教育网用户推荐台湾的源（光纤就是爽啊～～）更新源列表及系统的命令为：

sudo apt-get update
sudo apt-get dist-upgrade

更新系统完毕后，安装中文语言包，到系统(System)->系统管理(Administration)->语言支持(Language Support)中，勾选汉语(Chinese)选项，自动下载中文语言包和SCIM的拼音支持。一直很奇怪hardy的语言包依赖关系，居然会把hunderbird安装上- -b

如果是NVIDIA/ATI显卡，可以安装显卡驱动了，在系统->硬件驱动中，勾选“未使用”的显卡驱动，会自动从源上下载相关驱动。重启后就可以正常使用特效等功能了。如果在“硬件驱动”中还有其他“未使用”的驱动，也可以勾选下载安装，通常无线网卡和某些声卡。

★软件包安装

系统相关

QT程序输入法: scim-bridge-client-qt, scim-bridge-client-qt4

Compiz特效管理器: compizconfig-settings-manager

转换ID3v2标签为UTF-8: python-mutagen

★转换当前目录层下的mp3文件：

mid3iconv -e gbk *.mp3

★转换当前目录下包括子目录在内的所有 mp3：

find . -iname "*.mp3" -execdir mid3iconv -e gbk {} ;

命令集合：

sudo apt-get install scim-bridge-client-qt scim-bridge-client-qt4
compizconfig-settings-manager python-mutagen

编译器相关

gcc: build-essential

★因为gcc版本的问题，build-essential中不包含libstdc++5了，在安装一些deb或者从源码编译的时候会出错，需要自己安装。

java: sun-java6-*

editor: vim-full

★修改配置文件：

cp /etc/vim/vimrc ~/.vimrc
vi ~/.vimrc

可以Google一下配置文件，网上有很多强大的配置文件。

命令集合：

sudo apt-get install libstdc++5 gcc-3.3-base build-essential sun-java6-* vim-full

多媒体相关

解码器: w32codecs, gstreamer

Video: vlc, smplayer

★如果Mplayer播放出现异常，将视频驱动选择为x11再试试。在hardy中的mplayer打开中文文件名的BUG已经消除了。

Audio: amarok, audacious

命令集合：

sudo apt-get install gstreamer0.10-* smplayer mplayer-nogui mozilla-mplayer
vlc mozilla-plugin-vlc vlc-plugin-esd audacious audacious-plugins audacious-plugins-extra amarok
wget -c http://www.debian-multimedia.org/pool/main/w/w32codecs/w32codecs_20071007-0.1_i386.deb
sudo dpkg -i w32codecs_20071007-0.1_i386.deb

网络相关

FTP: gftp

★解决gFTP显示中文列表乱码问题：

修改启动脚本文件：

sudo gedit /usr/bin/gftp

在#!/bin/sh之后加上一行：

export LANG=zh_CN

保存并执行命令生成相应的locale：

sudo locale-gen zh_CN

gFTP已经落伍了，推荐Filezilla，在设置里可以调整locale，下载中文文件也不会乱码，还能登陆ipv6站点~

Web Server: LAMP

ssh: OpenSSH

Gmail邮件监测: checkgmail

命令集合：

sudo apt-get install filezilla checkgmail
openssh-server apache2 mysql-server php5-common php5-mysql

其他软件

解压缩: rar压缩&解压缩、rpm包转换工具、7zip

电子书: kchmviewer

pdf: Adobe Reader

★到此处下载：http://www.adobe.com/products/acrobat/readstep2_allversions.html

字典: 星际译王stardict

★字典库需要到http://stardict.sourceforge.net/cn/Dictionaries_zh_CN.php下载，下载完毕后解压缩到/usr/share/stardict/dic

命令集合：

sudo apt-get install rar unrar alien p7zip-full kchmviewer stardict
